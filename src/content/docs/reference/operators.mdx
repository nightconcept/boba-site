---
title: Operators
---

Boba supports a variety of operators.

## Relational (binary)

```boba
// eq    lt   gt     le    ge
   ==    <     >     <=    >=
```

## Not equal

```boba
!=
```

## Arithmetic (binary)

```boba
//add    sub    mul    div    mod    pow
   +      -      *      /      %      ^
```

## Arithmetic (unary)

```boba
++
--
```

## Logic

```boba
null and false --> null
false and null --> false
not true --> false
0 and 20 --> 20
10 and 20 --> 20
